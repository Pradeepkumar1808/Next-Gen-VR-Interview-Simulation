🖥️ System Requirements

Minimum Requirements

- OS: Windows 10 / macOS (latest supported)
- RAM: 8 GB
- Processor: Intel i5 / Ryzen 5 or above
- GPU: Integrated or Dedicated GPU (recommended: GTX 1650 / equivalent)
- Storage: 2–5 GB free space

Recommended (For Smooth VR Experience)

- RAM: 16 GB
- GPU: RTX 2060 / above
- VR Headset (Optional): Oculus / HTC Vive

⚙️ Installation Steps

For Alpha Build (Users)

1. Download the files from the Alpha Build link
2. Extract the ZIP file (if compressed)
3. Open the folder:
   - Windows: Run ".exe" file
   - Mac: Open ".app" file (allow permissions if blocked)
4. Wait for the application to load

---

For Beta Project File (Developers)

1. Download the Beta Test File
2. Open Unity Hub
3. Click Add Project
4. Select the extracted project folder
5. Open using the recommended Unity version
6. Click Play to test inside Unity

🎮 How to Use the Application

1. Launch

- Start the application
- You will enter the Main Menu

2. Menu Options

- ▶️ Start Interview
- 🔁 Restart
- ❌ Exit

3. Inside Interview Scene

- You will be placed in a virtual interview room
- Sit/stand facing the interviewer
- Questions will be asked based on selected domain (CSE-focused)

4. Controls

Keyboard & Mouse

- "W A S D" → Movement
- Mouse → Look around
- "E / Click" → Interact

VR Mode (if enabled)

- Use controller triggers for interaction
- Look-based or hand-based selection

⚠️ Troubleshooting

App Not Opening

- Run as Administrator (Windows)
- Allow app in Security Settings (Mac)

No Audio

- Check system sound output
- Ensure correct audio device is selected

Lag / Low FPS

- Lower graphics settings
- Close background apps

VR Not Working

- Ensure headset is connected properly
- Launch via VR-supported runtime (SteamVR / Oculus)
